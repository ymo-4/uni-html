Списки

1. Маркировані списки (ul)

Стандартні маркери:

list-style-type: disc;

  • Елемент з кружечком
  • Другий елемент
  • Третій елемент

list-style-type: circle;

  • Порожній кружечок
  • Другий елемент
  • Третій елемент

list-style-type: square;

  • Квадратний маркер
  • Другий елемент
  • Третій елемент

list-style-type: none;

  • Без маркера
  • Другий елемент
  • Третій елемент

Маркери з зображеннями:

list-style-image: url('icons/arrow.png');

  • Стрілка як маркер (з папки icons/)
  • Другий елемент зі стрілкою
  • Третій елемент

background-image для кращого контролю

  • Зірочка з icons/star.png
  • Другий елемент з зірочкою
  • Третій елемент

Емоджі як маркери:

  • Емоджі через CSS :before
  • Не залежить від зображень
  • Завжди відображається

Позиція маркерів:

list-style-position: outside; (за замовчуванням)

  • Маркер поза текстом елементу списку, що створює відступ для всього тексту
  • Другий довгий елемент

list-style-position: inside;

  • Маркер всередині тексту елементу списку, тому текст обгортає маркер
  • Другий довгий елемент

2. Нумеровані списки (ol)

list-style-type: decimal;

  1. Звичайні числа
  2. Другий пункт
  3. Третій пункт

list-style-type: decimal-leading-zero;

  1. Числа з нулем
  2. Другий пункт
  3. Третій пункт

list-style-type: lower-alpha;

  1. Малі латинські літери
  2. Другий пункт
  3. Третій пункт

list-style-type: upper-alpha;

  1. Великі латинські літери
  2. Другий пункт
  3. Третій пункт

list-style-type: lower-roman;

  1. Малі римські цифри
  2. Другий пункт
  3. Третій пункт

list-style-type: upper-roman;

  1. Великі римські цифри
  2. Другий пункт
  3. Третій пункт

list-style-type: lower-greek;

  1. Грецькі літери
  2. Другий пункт
  3. Третій пункт

Кастомна нумерація через CSS counters:

  1. Кастомний формат нумерації
  2. Можна додати будь-який префікс
  3. Повний контроль над виглядом

3. Багаторівневі списки

Змішаний багаторівневий список:

  • Перший рівень (диск)
    • Другий рівень (коло)
      • Третій рівень (квадрат)
      • Ще один елемент третього рівня
    • Повернулися на другий рівень
  • Знову перший рівень
    1. Нумерований підсписок (a, b, c)
      1. Ще глибший рівень (i, ii, iii)
      2. Другий елемент
    2. Повернулися на рівень вище

Кастомний багаторівневий зі стилізацією:

  • Головна категорія
    • Підкaтeгoрія 1
      • Деталь 1
      • Деталь 2
    • Підкaтeгoрія 2
  • Друга головна категорія
    • Підкaтeгoрія A
    • Підкaтeгoрія B

4. Списки визначень (dl, dt, dd)

Стандартний список визначень:

HTML
HyperText Markup Language - мова розмітки гіпертексту, що використовується для створення веб-сторінок.
CSS
Cascading Style Sheets - каскадні таблиці стилів, що використовуються для опису зовнішнього вигляду документа.
JavaScript
Мова програмування, що виконується у браузері та додає інтерактивність веб-сторінкам.
Також може використовуватися на сервері (Node.js).
