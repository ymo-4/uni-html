Скорочення та абревіатури

В веб-розробці часто використовуються скорочення:

  • HTML — мова розмітки
  • CSS — каскадні таблиці стилів
  • JS — мова програмування
  • API — програмний інтерфейс
  • SEO — оптимізація для пошукових систем

Наведіть курсор на скорочення, щоб побачити повну назву.

Тестування адаптивності

Змініть розмір вікна браузера або відкрийте сторінку на мобільному пристрої для перевірки адаптивності.

Використані теги:

  • <q> — короткі цитати
  • <blockquote> — блочні цитати
  • <abbr> — абревіатури
  • <address> — контактна інформація
  • <cite> — джерела цитат
  • <bdo> — напрямок тексту
  • <dfn> — визначення термінів
  • <code> — код
  • <kbd> — клавіши
  • <samp> — вивід програми
  • <var> — змінні
  • <mark> — виділення
  • <del> — видалений текст
  • <ins> — вставлений текст
  • <sub> — нижній індекс
  • <sup> — верхній індекс

Медіа-запити:

  • Мобільні: <= 480px
  • Планшети: <= 768px
  • Десктоп: > 768px
  • Великі екрани: >= 1200px
  • Друк: спеціальні стилі
